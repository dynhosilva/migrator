# Branding

## Paleta de cores

| Nome | Hex | Uso |
|---|---|---|
| Primary | `#6366f1` | Ações principais, badges |
| Success | `#22c55e` | Fases concluídas, `safeToMigrate: true` |
| Warning | `#f59e0b` | Avisos, confidence low |
| Error | `#ef4444` | Erros, fases falhas, `safeToMigrate: false` |
| Info | `#3b82f6` | Informações neutras |
| Muted | `#6b7280` | Texto secundário |
| Background | `#0f172a` | Fundo da TUI |

## Tipografia

- **Títulos:** monospace bold — compatível com todos os terminais
- **Corpo:** monospace regular
- **Destaques:** `chalk.bold` / `chalk.cyan` / `chalk.green`

## Ícones e símbolos (ASCII-safe)

| Símbolo | Significado |
|---|---|
| `✓` | Concluído |
| `✗` | Falha |
| `⚠` | Aviso |
| `ℹ` | Info |
| `→` | Navegação/fluxo |
| `◆` | Fase ativa |
| `◇` | Fase pendente |
| `●` | Running |

## Logo

O logotipo textual do projeto:

```
lovable-migrate
```

Em contextos ASCII (terminal, README):
